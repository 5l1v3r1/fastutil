Provides type-specific classes for integer elements or keys.

Note that because of the clash with {@link java.util.List#remove(int)}, the type-specific removal method of an {@link it.unimi.dsi.fastutil.ints.IntCollection} is {@link it.unimi.dsi.fastutil.ints.IntCollection#rem(int)}. This is fixed for sets by reintroducing {@link it.unimi.dsi.fastutil.ints.IntSet#remove(int)}, but, for instance, if you want to remove an element from a list you will need to use rem().
